Projet Réalité Virtuel

Le projet Le Voyage Temporel est un projet conçu dans un cadre étudiant. Le concept est un Escape game auquel le joueur sera poussé à observer le lieu pour comprendre comment passer de niveau. À chaque niveau, le joueur va dans une époque différente que celle précédente jusqu'à l'apocalypse.! Ce projet a été fait en équipe de deux. Maïka Désy et moi avons travailler d'arrache-pied pour ce résultat Le grand défi consistait à la séparation des tâches. Nous devions rester organisé et structurer pour éviter de corrompre le travail de l'autre.


Notre plan d'organisation consistait à se donner nos scènes respectives du jeu et aussi chacun avait sa scène de test pour faire des tests de code pendant que l'un travail dans les vrais scènes.



Temps de production : 64 heures
